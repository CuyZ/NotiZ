Installation
============

Via Composer
------------

``$ composer require cuyz/notiz``

Via the TER
-----------

Download the extension at: https://extensions.typo3.org/extension/notiz/

Extension manager
-----------------

Once the extension is available on the system, install it in the Extension
Manager:

.. figure:: /Images/02-Installation/extension-manager.png
    :alt: NotiZ installation in the Extension Manager

-----

You're now ready to go!

Have a preview of the extension features in the chapter “:ref:`example`”.
